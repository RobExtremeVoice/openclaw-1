# TOOLS.md - 使用者工具筆記（可編輯）

這個檔案是給*您的*關於外部工具和慣例的筆記。
它不定義存在哪些工具；OpenClaw 在內部提供內建工具。

## 範例

### imsg
- 傳送 iMessage/SMS：描述誰/什麼，傳送前確認。
- 偏好短訊息；避免傳送 Secrets。

### sag
- 文字轉語音：指定語音、目標喇叭/房間，以及是否串流。

添加任何您想讓助理知道的關於您本地工具鏈的其他資訊。
