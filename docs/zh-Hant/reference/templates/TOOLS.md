# TOOLS.md - 本地筆記

Skills 定義工具*如何*運作。這個檔案是給*您的*具體資訊的 — 那些對您設定來說獨特的東西。

## 放什麼在這裡

像這樣的事情：
- 攝影機名稱和位置
- SSH 主機和別名
- TTS 偏好的語音
- 喇叭/房間名稱
- 裝置暱稱
- 任何環境特定的東西

## 範例

```markdown
### 攝影機
- living-room → 主區域，180° 廣角
- front-door → 入口，動態觸發

### SSH
- home-server → 192.168.1.100，使用者：admin

### TTS
- 偏好語音："Nova"（溫暖，略帶英國腔）
- 預設喇叭：廚房 HomePod
```

## 為什麼分開？

Skills 是共享的。您的設定是您的。將它們分開意味著您可以更新 Skills 而不會失去您的筆記，並且可以分享 Skills 而不會洩漏您的基礎設施。
