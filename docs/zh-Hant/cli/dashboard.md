# `openclaw dashboard`

使用目前的認證資訊開啟控制介面 (Control UI)。

```bash
# 開啟控制介面
openclaw dashboard

# 僅輸出介面網址而不實際開啟瀏覽器
openclaw dashboard --no-open
```
